3.神经网络设计
原始图像尺寸为 [Batch,3,64,64]
第1卷积层
使用16个3×3卷积核提取初级纹理。卷积后尺寸不变为 [Batch,16,64,64]；经过最大池化后空间分辨率减半，尺寸变为 [Batch,16,32,32]
第2卷积层 
通道数加深至32。卷积后为 [Batch,32,32,32]；二次池化后尺寸变为 [Batch,32,16,16]
第3卷积层
通道数扩充至64以整合高阶语义。卷积后为 [Batch,64,16,16]；三次池化后尺寸变为 [Batch,64,8,8]。

特征展平
将多维特征图打平为一维向量，作为分类器的输入，尺寸计算：64×8×8=4096，转换后尺寸为[Batch,4096]

全连接映射层
通过一个线性层将特征从4096维压缩至128维进行语义融合。输出尺寸为[Batch,128]

分类输出层
最终的线性映射，直接输出3个分类（红、黄、蓝锥桶）的预测置信度。输出尺寸为[Batch，3]
